PhD projects

1 - I examine the consumption profile of individuals in various economic activities at the municipal level, using card transaction data and segmenting the data by credit card profile, customer and economic activity. I use the private database, which is one of the largest banks in Latin America. Additionally, explore this data using Big Data techniques and programs in R, Python and Spark to effectively deal with complex data sets.


2 - A second database that encompasses all payments methods. The main objective is to identify, in real time, the sectors most vulnerable to adverse climate shocks, providing a detailed understanding of how extreme weather events can affect economic dynamics. A topological network analysis will be carried out to discover significant patterns and clusters, which facilitates understanding of how payment flows are organized both spatially and sectorally.

Programming Languages: R, Spark and Python
