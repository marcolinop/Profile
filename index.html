PhD projects

1 - I examine the consumption profile of individuals in various economic activities at the municipal level, using card transactions data and segmenting the data by credit card profile, customer and economic activity. I use the private database, which is one of the largest banks in Latin America. Additionally, explore this data using Big Data techniques and programs in R, Python and Spark to effectively deal with complex data sets.


2 - A second database that includes all payment methods. The main objective is to identify, in real time, the sectors most vulnerable to adverse events (climate, economic and health crises), providing a detailed understanding of how extreme weather events can affect economic dynamics. A topological network analysis will be performed to discover significant patterns and clusters, which facilitates the understanding of consumer behavior.

Programming Languages: R, Spark and Python
Data Visualization: Power BI, spotfire
